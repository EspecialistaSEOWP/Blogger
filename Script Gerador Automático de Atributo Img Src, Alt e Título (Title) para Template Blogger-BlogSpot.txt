Editar os os atributos ALT e Title manualmente de uma imagem é fácil. Mas imagina editar 2.000 mil fotos? Sim, dá um baita trabalho, e quando se trata de melhorar o desempenho de SEO, é primordial otimizar as Tags ALT e Title para aumentar a relevância nas busca do Google Images. Agora, imagina se existisse um Script capaz de fazer toda otimização de imagens? Adicionar Tag ALT e Title com Auto Generator Image para Blogspot Blogger é possessível...

Sei que Blogger atualizou seus recursos e widgets, com isso, alguns scripts de plug-in deixaram de funcionar. Isso inclui alguns geradores de etiquetas auto alt para imagens. Mas, felizmente, estou compartilhando um que ainda funciona perfeitamente com não apenas o novo sistema, mas também com a antiga interface do Blogger, que naturalmente é o nosso tutorial caracterizado para hoje.

Siga as etapas abaixo para adicionar o gerador ao seu blog: Vá para a página Design no painel do Blogger e clique em Modelo. Clique em "Editar HTML" e depois em "Continuar". Procure o </ body> tag usando "Ctrl + F".

Copie / cole o código abaixo logo antes da tag </head>:

Salve o modelo.

Para verificar se o script está funcionando, basta abrir o seu blog e passe o cursor do mouse sobre a imagem. Em seguida, você verá o nome da imagem.

Adicionar tags "alt" e "title" a uma imagem pode ser bastante tedioso, especialmente se você estiver lidando com muitos deles. Portanto, geradores como este devem economizar muito tempo e energia. Além disso, torna o seu blog SEO-friendly também.

Como o script funciona:

Este script tomará automaticamente o nome da imagem e adicionará uma tag "alt" e "título" a todas as imagens do seu blog.

Este script não só adiciona um "alt" e "title" tags, mas também dá um sinal de mais '+' se houver espaço no nome da imagem enviada.

Portanto, tudo o que você precisa fazer para usar o script é certificar-se de que você nomear suas imagens corretamente antes de enviá-las para o Blogger. Caso contrário, não será possível ver as tags de título apropriadas.
